# 리더보드/대시보드 명세

## 1) 리더보드

- 기본 컬럼: rank, teamName, score, submittedAt
- 아이디어 해커톤 확장 컬럼: participant/judge score
- 아티팩트 링크: webUrl, pdfUrl, planTitle

## 2) 시각화

- 순위 막대 차트(Top N)
- 제출 시점 분포(타임라인)

- 점수 구성(참가자 30% / 심사위원 70%) 도넛 차트
- 개인/팀 이력 추세(대회별 성과)

## 3) 개인 대시보드

- 내가 참가한 대회 목록

- 제출물 기록(기획서/웹/PDF)
- 최종 순위/점수 스냅샷
- 회고 메모

- 수료증 보관함

## 4) 수료증 저장 기능

- 업로드(이미지/PDF) 또는 URL 등록

- 메타데이터: 대회명, 수여일, 발급기관, 설명
- 다운로드/외부 링크 복사

## 5) 면접용 출력

- 요약 카드 생성: 대회, 역할, 결과, 링크
- 공개용 페이지 토글(향후)
