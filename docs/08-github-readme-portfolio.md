# GitHub README -> 포트폴리오 HTML 생성

## 1) 플로우

1. 사용자가 GitHub 계정 입력
2. 대표 저장소 선택(또는 pinned repo 자동 수집)
3. README.md 수집
4. Markdown AST 파싱
5. UI 템플릿 매핑
6. HTML 단일 파일 생성/다운로드

## 2) 파싱 기준

- 제목/소개
- 기술 스택
- 주요 프로젝트
- 실행 방법
- 스크린샷/링크
- 라이선스

## 3) 누락 정보 보완

- 프로젝트 역할
- 문제/해결/성과
- 정량 지표(속도, 사용자수, 비용절감 등)

## 4) 출력 형식

- `portfolio.html`
- 테마: 기본(토스 무드)
- SEO 메타 태그 자동 생성

## 5) 보안/제한

- 공개 저장소 README만 기본 지원
- 사용자 토큰 저장 금지
- 외부 링크는 새 창 + noopener 처리
