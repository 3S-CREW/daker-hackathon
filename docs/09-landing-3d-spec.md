# 랜딩 3D 인터랙션 명세

## 1) 목표

- 첫 인상에서 "해커톤/빌드/성과" 메시지를 강하게 전달.

## 2) 연출 요소

- Framer Motion 기반 카메라/카드 깊이감
- 마우스/터치 반응형 패럴랙스
- CTA 버튼 2~3개에 집중

## 3) 기술 선택

- 기본: Framer Motion + CSS 3D transform
- 선택 확장: react-three-fiber(성능 검증 후)

## 4) 접근성/성능 기준

- `prefers-reduced-motion` 대응 필수
- 저사양 기기에서는 2D fallback
- LCP 방해 금지(핵심 텍스트 우선 렌더)

## 5) 금지사항

- 과도한 파티클/무거운 텍스처 금지
- 첫 화면에서 의미 없는 자동 재생 사운드 금지
